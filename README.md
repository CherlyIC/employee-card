
# Employee Directory
A simple react app that fetch and displays employee data

## Features
- fetches data from JSONPlaceholder API
- real time search by name and email
- print and download employee cards
- fully responsive layout

## technologies

- react
- tailwind css
- html-to-image
- DiceBear API

## How to run

1. clone the repository
   https://github.com/CherlyIC/employee-card.git

2. Install dependencies
  `npm install`

3. run the server

   `npm run dev`

## API Used

https://jsonplaceholder.typicode.com/users
